# CSS（Cascading Style Sheets，层叠样式表）

## 创作背景
  就像调色板能将白纸变为艺术品，CSS能将枯燥的HTML文本变为视觉盛宴。`CSS`是一种样式表语言，用于描述`HTML或XML`（包括`SVG`）文档的呈现方式。
  核心思想：实现内容与表现的分离
  - `HTML`负责定义网页的结构和内容
  - `CSS`负责控制网页的外观和布局
  没有`CSS`，网页将只是枯燥的、堆砌在一起的文本和链接。`CSS`是构建美观、用户体验良好的现代网页的基石。
